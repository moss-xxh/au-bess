**问题描述**: 用户从首页点击"立即前往"后，会自动跳过登录页面
**根本原因**: login.html第365-374行存在自动检查session并跳转的逻辑

**问题代码定位**:
```javascript
// pages/login.html 第365-374行
var session = JSON.parse(localStorage.getItem('au002_session') || '{}');
if (session.token && session.role) {
    // 自动跳转到对应dashboard
} else if (session.token) {
    window.location.href = '/pages/role-select.html';  // ❌ 问题所在
}
```

**工业级要求**: 储能系统需要强制二次身份确认，不能自动跳过登录
